# Brazflow Frontend

Esta é a interface de usuário do Brazflow, construída para fornecer uma experiência interativa e amigável para modelagem hidrológica.

## Funcionalidades

-   **Mapa Interativo**: Selecione bacias hidrográficas diretamente no mapa.
-   **Visualização de Resultados**: Explore simulações com gráficos de série temporal, curvas de permanência e outras métricas.
-   **Upload de Shapefiles**: Faça o upload de suas próprias áreas de bacia para simulação.

## Tecnologias

-   **Framework**: React com TypeScript
-   **Build Tool**: Vite
-   **Estilização**: Tailwind CSS
-   **Roteamento**: React Router
-   **Gerenciamento de Estado**: TanStack Query (React Query)
-   **Mapas**: OpenLayers
-   **Gráficos**: Recharts

## Desenvolvimento

Para detalhes completos sobre a configuração do ambiente, consulte o [**Guia de Desenvolvimento**](../DEVELOPMENT.md) na raiz do projeto.

### Instalação

1.  Certifique-se de ter Node.js e npm instalados.
2.  Instale as dependências:
    ```bash
    npm install
    ```

### Executando Localmente

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```
